# Tarea: Listas en Python / Incursión por Git HUb 

Este repositorio contiene una serie de funciones básicas para manipular listas (arreglos) en Python como parte de la tarea de Git y GitHub.

## Funciones implementadas:
1. **double(arr)**: Retorna una nueva lista con los valores duplicados.
2. **maximum(arr)**: Encuentra el valor más alto.
3. **average(arr)**: Calcula el promedio aritmético.
